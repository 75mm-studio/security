# 패스워드 정책
패스워드 정책입니다.

MPAA 관련규약 : DS-8.0, DS-8.1, DS-8.3, DS-11.5, AS-2.0~AS-2.10

## 클라우드(사용자)
- 대문자, 소문자, 숫자, 특수문자가 포함된 8자 이상 암호 사용.
- MFA(이중인증 시스템) 활성화

## 클라우드(클라이언트)
- 최초 암호를 부여하고 1회 로그인 이후 사용자가 암호를 변경하도록 설정
- MFA(이중인증 시스템) 활성화


## 인트라넷
- 대문자, 소문자, 숫자, 특수문자가 포함된 8자 이상 암호 사용.

## 지문인식장비
- 관리자의 지문을 등록합니다.
- 사용자의 지문을 등록합니다.
- 패스워드 정책은 위와 동일합니다.
